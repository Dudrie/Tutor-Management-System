Roles

Description

The Tutor-Management-System comes with several roles a user can take. Each role allows the user specific actions and disallows others. A user can have more than one role. You can find a complete description of all roles and which pages they can access in this document. Please note that all roles have access to the Login and Dashboard pages and therefore those are not mentioned in the lists of the roles.

ADMIN

Is allowed to take any action and can manage the settings of the TMS. The admin is also responsible for creating users, tutorials and scheincriterias.

Admin with tutorials

An admin itself can not be assigned to a tutorial. If an admin user should be assigned to a tutorial please assign the TUTOR role to that user aswell.

The same is true for admin users which also act as correctors for a tutorial (they need the CORRECTOR role).

Page Access

He/she has access to the following pages:

  • User Management
  • Tutorial Management
    note

    An admin can access pages inside a tutorial (like the Sheet Gradings) by clicking on "Manage" on a tutorial in the Tutorial Management.

  • Hand-Ins Management
  • Student Overview
  • Attendances
  • Scheincriteria Management
  • Settings

TUTOR

Only TUTOR users can be assigned to tutorials as tutors. Furthermore, only this type of user can be used as substitutes for tutorials.

Page Access

Tutors have access to the following pages. Pages marked with are also accessible by a substiute tutor.

  • Attendances
  • Presentations
  • Sheet Gradings
  • Scheinexams
  • Student Management
  • Team Management
  • Substitutes

CORRECTOR

A CORRECTOR user can be assinged as a corrector of a tutorial (or multiple). The user has access to all pages related to assigning gradings to students.

Page Access

  • Sheet Gradings
  • Scheinexams

EMPLOYEE

An EMPLOYEE user can manage a few internal things. It is used if there are employees who help the organization but should not have administrative priviliges.

Page Access

  • Tutorial Management
    note

    An employee user can not access the internals of a tutorial (except managing the substitutes).

  • Hand-Ins Management
  • Attendances
  • Scheincriteria Management
